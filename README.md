# NTP 时间同步工具 / NTP Time Sync Tool
一个轻量级、易用的 Windows 系统时间同步工具，支持自定义 NTP 服务器和定时自动同步功能。

A lightweight and easy-to-use Windows system time synchronization tool with support for custom NTP servers and scheduled auto-sync.

---

## ✨ 主要特性 / Key Features

- 🕐 **精确时间同步** - 通过 NTP 协议精确同步系统时间
- 🌐 **自定义服务器** - 支持使用任意 NTP 服务器地址
- ⏱️ **时间偏移补偿** - 可设置时间偏移量进行补偿调整
- ⏰ **定时自动同步** - 支持设置同步间隔（1-1440分钟），自动定期同步
- 🚀 **系统服务模式** - 可安装为 Windows 服务，开机自动启动
- 📱 **系统托盘运行** - 最小化到系统托盘，不占用任务栏
- 🌍 **中英文双语** - 完整的中英文界面支持

## 🚀 快速开始 / Quick Start

### 1. 运行程序 / Run the Program

右键 `ntp_time_sync.exe`，选择"以管理员身份运行"。

Right-click `ntp_time_sync.exe` and select "Run as administrator".

### 2. 配置设置 / Configure Settings

- **NTP 服务器** - 输入 NTP 服务器地址（默认：`pool.ntp.org`）
- **时间偏移** - 设置时间偏移量（如：`+60` 表示快 60 秒）
- **同步间隔** - 设置定时同步间隔（分钟）

### 3. 开始同步 / Start Syncing

- 点击"立即同步"按钮手动同步
- 或勾选"启用定时自动同步"启用自动同步

## 📝 使用说明 / Usage

### 基本功能 / Basic Functions

1. **立即同步** - 点击"立即同步"按钮执行一次时间同步
2. **定时同步** - 勾选"启用定时自动同步"并设置间隔
3. **系统服务** - 点击"安装服务"将程序安装为系统服务
4. **托盘操作** - 双击托盘图标恢复窗口，右键显示菜单

### 高级选项 / Advanced Options

- **启动后自动同步** - 程序启动时自动执行一次同步
- **同步后自动退出** - 同步完成后自动退出（适用于一次性同步）
- **自动禁用 Windows 时间同步** - 程序启动时自动禁用 W32Time 服务

## ❓ 常见问题 / FAQ

### Q: 为什么需要管理员权限？
A: 修改系统时间需要管理员权限，这是 Windows 系统的安全限制。

### Q: 杀毒软件报警怎么办？
A: 由于程序需要修改系统时间和创建服务，部分杀毒软件可能报警。这是正常现象，请将程序添加到白名单。

### Q: 支持哪些 NTP 服务器？
A: 支持所有标准的 NTP 服务器，如 `pool.ntp.org`、`time.windows.com` 等。
